\# UK Mass Transit Performance Dashboard



This project benchmarks Leeds bus routes against established UK tram networks.



\## How to Run

1\. \*\*Data Engineering\*\*: Run `01\_data\_engineering.R`. Note: You must update the `GTFS\_PATH` variable in the script to point to your local `itm\_all\_gtfs.zip`.

2\. \*\*Shiny App\*\*: Run `02\_build\_shiny\_dashboard.R` to launch the interactive dashboard.



\## Data Sources

\- Timetables via DfT Bus Open Data Service (BODS).
